## 🚀 Kurulum

### 📋 Gereksinimler
- Node.js `v16+`
- Discord Bot Token
- `npm` veya `yarn`

### 🛠️ Kurulum Adımları
1. Depoyu klonlayın:
   ```sh
   git clone https://github.com/HamsterMert/bos-altyapi.git
   ```
2. Gerekli bağımlılıkları yükleyin:
   ```sh
   npm install
   ```
3. `json/settings.json` dosyanızı yapılandırın.
4. Botu çalıştırın:
   ```sh
   node index.js
   ```
